# sql-data-analytics-project

A comprehensive collection of SQL scripts for data exploration, analytics, and reporting.  
These scripts cover core analytical tasks such as database exploration, measures and metrics, time-based trends, cumulative analytics, and segmentation.  
Each query focuses on a specific analytical use case and demonstrates best practices for writing clean and efficient SQL.
